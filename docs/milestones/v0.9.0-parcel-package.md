# Milestone: v0.9.0 Parcel Package

## Summary

This milestone promotes the parcel deliverable to a dedicated packaged module and removes the incomplete houseplan package from the publishable distribution.

## Scope

- package the finalized parcel ontology as `onto2ai_parcel`
- package the finalized parcel Neo4j constraint artifact
- package the parcel Neo4j query context
- package the parcel Pydantic model
- package the dataset-oriented parcel smoke test
- remove `onto2ai_houseplan` from the publishable package surface
- rebuild the distribution so published artifacts reflect the new package layout

## Delivered

- new package: `onto2ai_parcel/`
- packaged ontology: `onto2ai_parcel/ontology/Parcel.rdf`
- packaged artifacts:
  - `onto2ai_parcel/staging/pydantic_parcel_model.py`
  - `onto2ai_parcel/staging/neo4j_query_context.md`
  - `onto2ai_parcel/staging/neo4j_constraint.cypher`
  - `onto2ai_parcel/staging/parcel_schema_smoke_test.py`
- package path exports in `onto2ai_parcel/artifacts.py`
- package README in `onto2ai_parcel/README.md`
- packaging metadata updated in `setup.py` and `MANIFEST.in`
- incomplete `onto2ai_houseplan` removed from distribution

## Validation

- parcel ontology RDF syntax validation passed with `xmllint --noout`
- staged schema validation passed against `stagingdb`
- parcel dataset smoke test passed against `testdb`
- package module import checks passed for `onto2ai_parcel`
- source and wheel build completed successfully

## Version
